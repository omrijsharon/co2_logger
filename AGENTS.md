# AGENTS.md

## Implementation Behavior

When working on this project, follow these rules:

1. Do not stop after finishing a single milestone.
   Continue and complete as many remaining tasks as possible in the same run.

2. If progress is blocked by missing information, unclear requirements, or an ambiguous technical decision, stop and ask the user for clarification before continuing.
   The goal is to stay on the correct path rather than guessing when the direction is unclear.

3. It is acceptable to stop once the requested tasks are fully completed.

4. If you decide to stop for any reason, always report why you stopped.
   The reason must be explicit, even if the stop was intentional or appropriate.
